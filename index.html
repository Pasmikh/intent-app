Intent App

Description

This is an app for Android devices that replaces the default home screen. It requires you to record a voice message or type an intent to open any application, except for those marked in a special way. This intent is transcribed to text and then analyzed by an LLM (Large Language Model) to determine if the reason provided is valid.

The goal is to introduce a moment of reflection before opening potentially distracting apps like social media or messengers. To use these apps, you need to state a genuine reason.

Data Usage Policy

Your privacy is important. This application does not store any personal data that can be linked back to you as an individual.

The only data collected is:

  • The text derived from your voice or typed intents.
  • The names of the applications you request to open via an intent.
This information is used solely for the purpose of the LLM deciding whether to allow the app opening based on the stated intent. Specifically, OpenAI's Whisper model is used to transcribe voice recordings, and OpenAI's GPT-4o mini model is used to determine the validity of the intent. No other data is collected, and the collected data cannot be used to identify specific persons.
